Code for Generating Main and Supplementary Figures.

## How to use and reproduce the figures

1. All analysis scripts were written in R. The analysis environment was configured using both R and Python. Before running the code, please set up the required software environment according to the specifications provided in `analysis_sessionInfo.txt`.
2. The R Markdown (`.Rmd`) scripts are organized by figure and can be found in the `/R_scripts` directory.
3. Before running the scripts, you will need to prepare the required Seurat objects as input. These files can be downloaded from [Google Drive](https://drive.google.com/drive/folders/1B2yMuHGl4RVar_NYLMZuCUt0JENCyJxT?usp=sharing).
4. The `/output` folder contains all figures (PDF format) generated by the corresponding scripts and is provided for reference.
